## Getting Started

### Install

Clone project

```bash
git clone https://github.com/corners2wall/Rethink-Layout.git
```

Access the project directory.

```bash
cd Rethink-Layout
```

Install dependencies.

```bash
pnpm install
```

Serve with hot reload at <http://localhost:5173>.

```bash
pnpm run dev
```

### Lint

```bash
pnpm run lint
```

### Typecheck

```bash
pnpm run typecheck
```

### Build

```bash
pnpm run build
```

### Test

```bash
pnpm run test
```

View and interact with your tests via UI.

```bash
pnpm run test:ui
```
